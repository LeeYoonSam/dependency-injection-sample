## 1. Dependency Injection(DI)란?

Dependency Injection은 객체 지향 프로그래밍에서 객체 간의 의존성을 외부에서 주입하는 디자인 패턴입니다.DI를 사용하면 다음과 같은 이점이 있습니다:

- 코드의 재사용성 향상
- 테스트 용이성 증가
- 결합도 감소 및 유연성 증가
